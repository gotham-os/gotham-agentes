# Framework VSL (Video Sales Letter)

## Estrutura de 8 blocos

```
1. HOOK (0:00–0:30)
   → Para o scroll. Pergunta ou afirmação chocante.

2. PROMISE (0:30–1:00)
   → O que o espectador vai descobrir. O que vai mudar na vida dele.

3. STORY (1:00–3:00)
   → História do criador ou de um cliente. Identificação + transformação.
   → Arco: [Problema] → [Tentativas falhas] → [Descoberta] → [Resultado]

4. PROBLEMA AGRAVADO (3:00–4:00)
   → Por que o problema é pior do que parece. Consequências de não resolver.

5. SOLUÇÃO + MECANISMO ÚNICO (4:00–6:00)
   → O método/produto. Por que é diferente. O mecanismo único nomeado.

6. PROVA (6:00–8:00)
   → Depoimentos, dados, antes/depois, estudos.

7. OFERTA (8:00–10:00)
   → O que está sendo vendido. Stack de valor. Preço ancorado → preço real.
   → Bônus. Garantia. Urgência/escassez.

8. CTA (10:00–10:30)
   → Ação única. "Clique abaixo agora." Repetir urgência.
```

## Mini-VSL (60s para anúncio)

```
0:00–0:10  Hook (pergunta ou choque)
0:10–0:25  Problema + agitação rápida
0:25–0:45  Solução + mecanismo único nomeado
0:45–0:55  Prova social (número ou depoimento curto)
0:55–1:00  CTA direto
```
