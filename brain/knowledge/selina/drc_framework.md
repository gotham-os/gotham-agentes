# Framework DRC — Direct Response Copywriting

## Fórmula base: Dor → Agitação → Solução (PAS)

```
1. DOR    → identifica a dor específica do avatar
2. AGITA  → amplifica a consequência de não resolver
3. SOLUÇÃO → apresenta a oferta como saída inevitável
```

## Estrutura de oferta testável

| Elemento | O que é | Exemplo |
|---|---|---|
| Promessa | O que o comprador ganha | "Perca 5kg em 30 dias" |
| Prova | Por que acreditar | Depoimento + dados |
| Preço | Ancoragem + valor percebido | "R$997 → R$197 hoje" |
| Garantia | Remove risco | "30 dias ou devolvo" |
| Urgência | Por que agora | "Apenas 50 vagas" |
| Bônus | Eleva valor percebido | Bônus > preço psicológico |

## Mercados e tons

| Mercado | Tom | Gatilho principal |
|---|---|---|
| BR | Emocional, direto, familiar | Transformação, comunidade |
| ES (tier-1) | Formal, aspiracional, autoridade | Status, família, segurança |
| EN | Data-driven, prático, ceticismo alto | Prova social, ROI numérico |
