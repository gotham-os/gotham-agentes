# Frameworks de Pesquisa — Ra's al Ghul

## Hierarquia de evidência

```
Nível 1 (mais forte) → Dado primário: pesquisa própria, teste real, dado de plataforma
Nível 2             → Dado secundário: artigo acadêmico, relatório de setor, estudo publicado
Nível 3             → Dado terciário: notícia, post de especialista, fórum
Nível 4 (mais fraco) → Opinião não fundamentada, anedota, "ouvi dizer"
```

Sempre identificar o nível de evidência na resposta.

---

## Framework OODA para Intel de Mercado

```
O → Observar: coletar dados brutos (Reddit, Trends, Ads Library, ScienceDaily)
O → Orientar: contextualizar com o que já sabemos sobre o nicho
D → Decidir: qual hipótese é mais provável
A → Agir: recomendar próximo passo
```

---

## Checklist de pesquisa de nicho

- [ ] Tamanho do mercado (fonte + data)
- [ ] Top 3 players ativos (Meta Ads Library — quem está gastando)
- [ ] Dores primárias (Reddit, ReclameAqui, fóruns)
- [ ] Palavras que o avatar usa para descrever a dor (verbatim — não parafrasear)
- [ ] Tendência de volume de busca (Google Trends 12 meses)
- [ ] Gaps: o que o mercado não oferece bem
- [ ] Evidência de disposição a pagar (preços praticados, GMV de produtos similares)
